# Employee Management Web Application

Welcome to the Employee Management Web Application! This application is designed to efficiently handle employee details and salary management. Built upon Object-Oriented Programming (OOP) principles and utilizing a MySQL database, it provides a streamlined interface for managing employee information.

## Key Features:

- **Single Interface:** The application offers a unified interface for performing CRUD (Create, Read, Update, Delete) operations on employee records.
- **File Upload Capability:** Users can conveniently upload employee images or documents directly within the application.
- **Database Setup:** Prior to running the application, ensure that the database file named "employee_test" provided in the project folder is properly stored.
- **Project Start Path:** Access the application by navigating to `<base url>/Employee-management` in your web browser.

## Getting Started:

1. **Database Setup:**
   - Locate the database file named "employee_test" within the project folder.
   - Before running the application, ensure that this database file is appropriately stored.

2. **Accessing the Application:**
   - Launch your web browser.
   - Enter `<base url>/Employee-management` in the address bar to access the Employee Management interface.

3. **Utilizing the Application:**
   - Once logged in, you'll have access to various functionalities for managing employee details and salary information.
   - Explore the intuitive interface to perform CRUD operations on employee records and upload images or documents as needed.
